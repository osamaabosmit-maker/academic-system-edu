# 🏆 نظام Edu-AI للتقييم الأكاديمي الشامل
### الجامعة الأسمرية الإسلامية | كلية التربية زليتن - قسم الحاسوب

هذا التطبيق هو منصة رقمية متكاملة تهدف إلى أتمتة عملية التصحيح والتقييم الأكاديمي، مما يسهل عمل عضو هيئة التدريس ويضمن دقة النتائج للطلاب.

---

## ✨ المميزات الرئيسية
* **معالجة الملفات:** قراءة آلية لأسئلة الاختبارات من ملفات Word (DOCX).
* **التصحيح الذكي:** مقارنة فورية بين إجابات الطلاب والإجابة النموذجية.
* **النتائج الفورية:** احتساب النسبة المئوية والتقدير العام (ممتاز، جيد جداً، إلخ) فورياً.
* **التفاعل البصري:** واجهة مستخدم حديثة تدعم اللغة العربية (RTL) مع تأثيرات بصرية عند النجاح.

## 🛠️ المتطلبات التقنية
لتشغيل المشروع، يجب توفر مكتبات Python التالية:
* `streamlit`
* `python-docx`
* `qrcode`

## 🚀 طريقة التشغيل
1. قم بتحميل المستودع (Clone) على جهازك.
2. ثبت المكتبات المطلوبة:
   `pip install -r requirements.txt`
3. قم بتشغيل التطبيق عبر الأمر:
   `streamlit run app.py`
